Bar Graph Plot

Here is an example of a bar graph:

Here, data is read from an ASCII file at a URL. The format of that file is the "old ptplot format," which is simpler than PlotML, but not as compatible with modern web tools. The horizontal axis is manually labeled with tick marks indicating years. It is possible to control the thickness of bars and the offset of the bars behind by editing that file. For details about the file format, see the documentation for the Plot and PlotBox classes.
